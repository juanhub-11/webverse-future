This website is made in VSCODE, using HTML, CSS AND JAVASCRIPT. The design prioritizes minimalism and negative space. By using a deep midnight-blue palette 
 paired with neon cyan accents, the site achieves a future look that emphasizes content over mess.
 "Smart School 2040" is a conceptual web experience designed for the Webverse competition.
 It explores a future where education is no longer confined to physical boundaries, but is an immersive, AI-driven journey.
 The project focuses on a high-end, futuristic effect known as Glassmorphism,
 combined with smooth animations to simulate a high-tech website.
